➕ Додати staged лістинг вручну
Використовується коли продукт виставлений вручну на DigiKey Marketplace і очікує затвердження — але API DigiKey ще не повертає його в списку продуктів. Введіть SKU товару зі складу Minerva, і буде створено запис лістингу зі статусом ⏳ Очікує затвердження.

ℹ️ Чому API не бачить staged продукти?
DigiKey API endpoint GET /products повертає тільки затверджені (опубліковані) продукти. Продукти що очікують перевірки DigiKey недоступні через API — тільки через веб-портал. Після затвердження DigiKey, наступна авто-перевірка (Авто-перевірка статусу) автоматично оновить статус на ✅ Опубліковано.
